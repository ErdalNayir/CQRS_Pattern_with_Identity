# CQRS_Pattern_with_Identity

The project aims to learn the CQRS pattern and Identity Server.

This project includes
*  Identity Server
*  Clean Code Architecture
*  Fluent Validation
*  Fluent API
*  Entity Framework
*  Code First Approach
*  CQRS Desing Pattern
*  AutoMapper
*  Jwt Token Validation
*  MediatR
